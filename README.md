# DamConvert

**Le convertisseur de fichiers universel — 100 % local, sans abonnement.**

DamConvert convertit images, audio, vidéo, documents, tableurs, ebooks, polices et archives directement sur votre PC. Aucun fichier n'est envoyé sur internet.

> Édité par **DamLab** · [lelabodedam.com](https://lelabodedam.com)

## ✨ Ce que DamConvert sait faire

- 🖼️ **Images** (80 formats, dont RAW d'appareil photo, HEIC, AVIF) → PNG, JPG, WEBP, PDF, fusion en 1 PDF, OCR…
- 🎧 **Audio** (30 formats) → MP3, WAV, FLAC, OGG, OPUS, AAC…
- 🎬 **Vidéo** (29 formats) → MP4, MKV, AVI, WEBM, GIF, extraction audio…
- 📄 **Documents** : PDF ↔ Word ↔ Texte, Markdown / HTML / RTF
- 📊 **Tableurs** : Excel / CSV → Excel, CSV, PDF
- 📚 **Ebooks** : EPUB, XPS, CBZ → PDF / Texte / Images
- 🔤 **Polices** : TTF / OTF / WOFF / WOFF2
- 🗜️ **Archives** : ZIP, 7Z, TAR…

Sélection multi-fichiers (même depuis plusieurs dossiers), conversion par lot, sortie dans le dossier de votre choix.
